# micro_game_on_raspberry

Проект представляет собой простую игровую систему на базе Raspberry Pi Pico с использованием библиотеки pibody.
В системе реализовано меню и две игры: Snake и упрощённый платформер в стиле Mario.

Управление осуществляется с помощью джойстика и одной кнопки:
	•	короткое нажатие — действие (запуск, прыжок, рестарт)
	•	удержание — выход в меню

Экран обновляется только при необходимости, что снижает мерцание и делает работу более комфортной.

⸻

⚙️ Возможности
	•	2 встроенные игры (Snake и Mario)
	•	управление через джойстик
	•	управление одной кнопкой
	•	оптимизированная отрисовка экрана
	•	звуковая обратная связь через буззер
